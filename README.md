# Laivanupotus

Pelaa tietokonetta vastaan laivanupotusta.

## komentorivitoiminnot

- testit:   mvn test
- testikattavuusraportti:   mvn jacoco:report
- suoritettavan jar:in generointi:   mvn package
- checkstyle:   mvn jxr:jxr checkstyle:checkstyle
